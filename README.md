# DotaHero

Небольшое клиент-серверное приложение на `Node.js + Express + Vanilla JS`, которое показывает всех героев Dota 2 в виде карточек, позволяет выбрать героя кликом или кнопкой `Рандом`, а затем показывает только выбранного героя.

## Запуск

1. Установить зависимости:

   ```bash
   npm install
   ```

2. Один раз синхронизировать локальные данные и картинки героев:

   ```bash
   npm run sync:heroes
   ```

3. Запустить приложение:

   ```bash
   npm start
   ```

4. Открыть в браузере:

   [http://localhost:3000](http://localhost:3000)

## Полезные команды

npm test
